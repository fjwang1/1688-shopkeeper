# 搜索策略

## 目标

这个 skill 的搜索接口只负责粗筛，详情页校验才是最终口径。

## 查询词写法

- 保留平台：如“适用于拼多多平台”
- 保留类目：如“厨房用品”“厨房小工具”“锅铲”“勺铲”
- 保留模式：如“一件代发”“包邮”
- 避免把条件写得过死，否则接口容易回同一批商品

## 推荐做法

- 先写 1 条主搜索词
- 如果结果重复严重，再补 1 到 2 条变体搜索词
- 常见变体：
  - `厨房用品` -> `厨房小工具`
  - `包邮` -> `一件代发`
  - `厨具套装` -> `锅铲/汤勺/漏勺/饭勺`
  - `不粘锅专用` -> `硅胶厨具`

## 结果解释

- `search_price`: 搜索接口返回的粗筛价，不一定是真实单件售价
- `one_piece_price`: 详情页出现 `1件包邮` 时对应的价格
- `multi_piece_price`: 详情页里 `≥2件` 或类似阶梯价
- `benefits`: 页面同时出现的权益文案，如 `官方仓退货`
